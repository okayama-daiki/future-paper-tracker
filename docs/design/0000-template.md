# 0000 Design Doc Template

Create this document when multiple design options are considered, regardless of the scope. This may include feature design, architecture decisions, library selection, or data model design.

## Context

This section describes the background of the design.
It should explain the problem being solved, the goals of the design, and any relevant constraints.

## Decision Log

List the design options that were considered and explain why the chosen option was selected.
For each decision, include the trade-offs and potential risks.

## Consequences

This section describes the consequences of the decisions, including trade-offs, risks, and implications for future development or implementation.
