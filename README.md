# Future Paper Tracker

A tracker for upcoming conference deadlines.

## About

Future Paper Tracker (FPT) is a tool designed to track upcoming conference deadlines, primarily for researchers in Japan.

Unlike many existing services that mainly focus on top-tier international conferences, FPT aims to cover a broader range of events, including:

- Domestic conferences held a few times per year
- Workshops and meetings without formal peer review
- Conferences that accept abstracts instead of full papers

The system is designed to aggregate and maintain conference information in a unified format.

Key features include:

- **Automatic discovery of conference information**
- Estimation of deadlines for conferences whose CFP has not yet been announced
